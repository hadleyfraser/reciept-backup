# Receipt Backup

Receipt Backup now includes a local loyalty card wallet for storing barcodes offline.

## Quick Start

- Open the project in Android Studio.
- Sync Gradle.
- Run the `app` configuration on a device or emulator.

## Loyalty Card Wallet

- Add, edit, delete, and reorder cards.
- Scan barcodes from the camera or a saved photo.
- View card details with a large barcode preview.

## Tests

Run unit tests from the project root:

```sh
./gradlew testDebugUnitTest
```
